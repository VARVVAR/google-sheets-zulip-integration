# Google Sheets to Zulip Notification Bot

Проект по автоматизации внутренней логистики: отправка уведомлений о готовности и отгрузке техники из Google Таблиц в мессенджер Zulip через API.

## 🚀 Основные функции
- **Автоматизация по событию:** Скрипт срабатывает мгновенно при установке чекбокса в таблице (триггер `onEdit`).
- **Интеграция с REST API:** Формирование и отправка POST-запросов к API Zulip.
- **Умная адресация:** Сопоставление фамилий из выпадающего списка в таблице с внутренними ID пользователей в мессенджере.
- **Интерфейсная логика:** После успешной отправки скрипт удаляет чекбокс, меняет текст на "✅ ОТПРАВЛЕНО" и применяет стилизацию к ячейке.

## 🛠 Технологический стек
- **JavaScript** (Google Apps Script)
- **REST API / JSON**
- **Google Sheets API**

## 💡 Особенности реализации
- Весь конфиг (ID колонок, URL, Email) вынесен в отдельные константы для удобства поддержки.
- Данные из строки считываются одним массивом (`getValues`), что оптимизирует скорость работы (минимизирует обращения к API Google Таблиц).
- Реализовано форматирование дат из объектов JS `Date` в читаемый строковый формат `dd.MM`.

## 📦 Как использовать
1. Создать проект в Google Apps Script, привязанный к таблице.
2. Скопировать код из `script.js`.
3. Настроить `ZULIP_CONFIG` (указать свои API-ключи).
4. Настроить `userMap` под своих сотрудников.
